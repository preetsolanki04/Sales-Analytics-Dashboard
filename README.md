# 📊 Sales Analytics Dashboard | Power BI, SQL & Excel

## 📌 Project Overview
This project is a **Sales Analytics Dashboard** created using **Power BI**, **SQL**, and **Excel**.  
The main goal of this project is to analyze sales data and generate meaningful business insights such as revenue trends, top products, customer performance, and region-wise sales.

This project demonstrates **end-to-end data analysis**, starting from raw data cleaning to final dashboard visualization.

---

## 🛠 Tools & Technologies Used
- **Power BI** – Data visualization & interactive dashboards  
- **SQL** – Data extraction, transformation, and analysis  
- **Microsoft Excel** – Data cleaning, preprocessing, and validation  

---

## 📂 Dataset
- Sales data containing:
  - Order details
  - Product information
  - Customer details
  - Region and sales amount
- Data source: CSV / Excel files (imported into Power BI)

---

## 🔄 Project Workflow
1. **Data Cleaning (Excel)**
   - Removed duplicates
   - Handled missing values
   - Standardized column names
   - Data formatting (dates, numbers)

2. **Data Analysis (SQL)**
   - Used SQL queries to:
     - Calculate total sales
     - Find top-selling products
     - Analyze monthly & yearly trends
     - Region-wise and customer-wise sales analysis

3. **Data Visualization (Power BI)**
   - Created interactive dashboards with:
     - Sales KPIs
     - Trend analysis
     - Category & region breakdown
     - Filters & slicers for better insights

---

## 📊 Key Dashboard Insights
- 📈 Total Sales & Revenue Trend
- 🏆 Top Performing Products
- 🌍 Region-wise Sales Performance
- 👥 Customer Contribution Analysis
- 📅 Monthly & Yearly Sales Trends
